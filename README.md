# Black Sea - Kraken Encounter / Siyah Deniz - Kraken Karşılaşması

## Description / Açıklama

**English:**
This is a simple text-based adventure game in Python, where the player navigates a ship on the black Sea and encounters a giant Kraken. The player can choose actions such as attack, defend or a risky heavy strike. The game demonstrates ** basic programming algorithms** including condiitonal statements ('İf/else'), loops ('while'), randomization, ad state updates (HP).

**Türkçe:**
Bu, python ile yazılmış basit bir metin tabanlı macera oyunudur. Oyuncu Siyah Deniz'de bir gemiyle ilerlerken dev bir kraken ile karşılaşır. Oyuncu saldırı, savunma veya riskil güçlü vuurş gibi eylemleri seçebilir. Oyun; **temel programlama algoritmalarını** gösterir: Koşullu ifadeler ('İf/else'), döngüler ('while'), rastgelelik (random) ve durum güncellemeleri (HP).
